# Asset Structure: Graphic Design Showcase

This folder contains the showcase MDX and the `works/` directory for individual creative works.
All media assets must be placed in the corresponding public media directory:
`public/api/media/graphic-design-showcase/`

## Creating a New Work

Inside `content/projects/graphic-design-showcase/works/[work-slug]/`, create a `metadata.json` file.
Then, create the media folder: `public/api/media/graphic-design-showcase/works/[work-slug]/`.

## Work Assets

Place the following files inside the media folder:

- `thumbnail.webp` (Required) - Static cover image (4:5, 1080x1350)

## Subdirectories

### `gallery/`

Place all high-resolution presentation files here.

- `01.webp`, `02.webp`, etc.
